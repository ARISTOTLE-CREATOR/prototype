# 💎 ArtisanGems

Luxury Jewelry Marketplace


## ✨ Overview

ArtisanGems is a premium jewelry e-commerce platform designed for handcrafted luxury collections.

---

## 🚀 Features

### 🏛 Customer Portal
- Secure login
- MFA Authentication
- Account management

### 💍 Jewelry Marketplace
- Rings
- Necklaces
- Earrings
- Bracelets
- Anklets
- Pendants

### 🤖 AI Concierge
- Personalized recommendations
- Gemstone guidance
- Style consultation

### 🔒 Security
- MFA
- Session management
- Customer vault

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
